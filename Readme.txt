# Στον φάκελο assignment μπορείτε να δείτε την εκφώνηση της εργασίας.

Μία από τις παραδοχές που κάναμε είναι ότι το κάθε αυτοκίνητο επιλέγει τυχαία τον κόμβο εξόδου του από τον
αυτοκινητόδρομο. Δηλαδή έναν κόμβο από τον 1ο μέχρι τον seg+1 
Ο αριθμός των διοδίων που δημιουργούνται σε κάθε είσοδο του εκάστοτε τμήματος επιλέγεται τυχαία, με μέγιστο μια σταθερά
MAX_TOLLS.

Για την μεταγλώτιση του προγράμματος υπάρχει το κατάλληλο makefile Και αρκεί να τρέξετε make στο directory του προγράμματος.

Διαθέσιμο test case είναι επίσης διαθέσιμο, εκτελόντας make run (στο make file μπορείτε να προσαρμόσετε το test case).

Αλλιώς, τρέχετε ./attiki_odos ARG1 ARG2 ARG3 ARG4 ARG5
Όπου τα arguments είναι με την σειρά που αναγράφεται στην εκφώνηση.

Τα δεδομένα αναπαρίστανται με τους εξής τρόπους:
-Οχήματα που βρίσκονται σε κάθε τμήμα, είσοδο κλπ: Λίστα (κομμάτια της λίστας μετακινούνται και ενοποιούνται με την λίστα του επόμενου κόμβου
όταν γίνεται μεταφορά οχημάτων από τμήμα σε τμήμα)
-Τα τμήματα είναι αποθηκευμένα σε έναν πίνακα με NSegs θέσεις
